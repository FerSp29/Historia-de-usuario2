# Sistema de Gestión de Inventario - Python 🐍

Este proyecto es una aplicación de consola desarrollada en *Python* que permite gestionar un inventario de productos de forma dinámica. El programa facilita el registro de artículos, la visualización de datos y el cálculo automático de estadísticas financieras del inventario.

---

## 🚀 Funcionalidades

El script Historia2.py ofrece las siguientes opciones:

1.  *Agregar Producto:* Registra el nombre, precio y cantidad. Los datos se almacenan en una lista de diccionarios.
2.  *Mostrar Inventario:* Despliega una lista organizada de todos los productos guardados. Si no hay productos, el sistema lo informa.
3.  *Calcular Estadísticas:* * Calcula el *Valor Total* del inventario (Precio × Cantidad).
    * Muestra el número de productos únicos registrados.
4.  *Menú Interactivo:* Un bucle continuo que permite realizar múltiples operaciones hasta que el usuario decida salir.

---

## 🛠️ Tecnologías y Conceptos Aplicados

* *Lenguaje:* Python 3.x
* *Estructuras de Datos:* Listas ([]) y Diccionarios ({}).
* *Control de Flujo:* Bucles while, condicionales if/elif/else y manejo de funciones.
* *Interacción:* Entrada y salida de datos por consola (input/print).
